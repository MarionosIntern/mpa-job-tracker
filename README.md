# mpa-job-tracker
 A fullstack job application tracker with Kanban Board and AI featues powered by CLaude API 

 ## Features

- **Kanban board** — drag jobs through Applied → Phone Screen → Interview → Offer → Rejected
- **Job Parser** — paste any job description, AI extracts skills, salary, culture signals, and match tips
- **Match Scoring** — resume vs. job description gap analysis with visual score ring
- **Cover Letter Generator** — tailored drafts with tone control, streamed in real time
- **Interview Prep** — role-specific questions with answer guidance


## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **AI**: Anthropic Claude API (streaming)
